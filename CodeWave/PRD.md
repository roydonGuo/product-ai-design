# 笔记转语音播客 App - 产品需求文档 (PRD)

## 一、产品概述

### 1.1 产品名称
**声记** - 笔记转语音播客应用

### 1.2 产品定位
一款将个人笔记智能转换为高质量语音播客的应用，让用户在通勤、运动、家务等场景下"听"笔记，解放双眼，高效获取知识。

### 1.3 产品愿景
成为用户的知识音频化助手，让每一篇笔记都能被"听见"。

---

## 二、目标用户

### 2.1 核心用户画像

| 用户类型 | 特征描述 | 核心需求 |
|---------|---------|---------|
| **知识工作者** | 25-40岁，经常记录学习笔记、工作笔记 | 通勤时收听笔记，高效复习 |
| **学生群体** | 18-25岁，大量课堂笔记、考试复习资料 | 碎片时间听笔记，加深记忆 |
| **内容创作者** | 自媒体、作家，有大量写作素材 | 听自己的文章/笔记，校对润色 |
| **视障人士** | 视力障碍群体 | 无障碍获取文字信息 |
| **运动爱好者** | 喜欢跑步、健身时学习 | 运动时收听知识内容 |

### 2.2 用户场景

```
场景1：通勤学习
小王每天地铁通勤1小时，他打开"声记"，收听昨天整理的《产品方法论》笔记，
到站时刚好听完，高效利用碎片时间。

场景2：运动复习
小李在备考CPA，每天晨跑时听自己的复习笔记，边运动边巩固知识点。

场景3：睡前收听
小张习惯睡前听自己的日记和随笔，在舒缓的声音中回顾一天，放松入睡。

场景4：家务陪伴
小陈周末做家务时，播放本周的学习笔记，不浪费时间。
```

### 3.2 核心功能详细说明

#### 3.2.1 笔记导入功能

| 功能点 | 优先级 | 描述 |
|-------|-------|------|
| 手动输入 | P0 | 支持直接在App内创建笔记 |
| 粘贴导入 | P0 | 支持从剪贴板粘贴文本 |
| 文件导入 | P0 | 支持TXT、Markdown、PDF导入 |
| 第三方同步 | P1 | 支持Notion、印象笔记、语雀、飞书文档同步 |
| OCR识别 | P2 | 支持图片文字识别导入 |

| 功能点 | 优先级 | 描述 |

| 语调调节 | P1 | 支持语调高低调节 |
| 情感风格 | P1 | 专业/活泼/舒缓/新闻播报等风格 |
| 多角色配音 | P2 | 对话类笔记支持不同角色配音 |
| 背景音乐 | P2 | 支持添加背景音乐 |

#### 3.2.3 播放功能

| 功能点 | 优先级 | 描述 |

| 进度控制 | P0 | 拖动进度条、快进快退 |
| 倍速播放 | P0 | 0.5x/0.75x/1.0x/1.25x/1.5x/2.0x |
| 定时关闭 | P0 | 睡眠定时器，支持定时关闭 |
| 后台播放 | P0 | 支持后台播放、锁屏播放 |
| 播放列表 | P0 | 创建、管理播放列表 |
| 章节跳转 | P1 | 长笔记支持章节跳转 |
| 书签功能 | P1 | 标记重要位置，快速跳转 |
| 离线下载 | P1 | 支持下载音频离线播放 |

#### 3.2.4 音频库管理

| 功能点 | 优先级 | 描述 |

## 四、非功能需求

### 4.1 性能要求

| 指标 | 要求 |
|-----|------|
| 语音转换速度 | 1000字以内笔记，转换时间<10秒 |
| 播放启动时间 | <1秒 |
| 应用启动时间 | <2秒 |
| 音频文件大小 | 1分钟音频<500KB（压缩后） |

### 4.2 兼容性要求

- **iOS**: iOS 13.0及以上
- **Android**: Android 8.0及以上
- **Web端**: 支持主流浏览器

### 4.3 安全性要求

- 用户数据加密存储
- 音频文件本地加密
- 支持应用锁（指纹/面容）
- 隐私笔记保护

### 4.4 可用性要求

- 支持深色模式
- 支持无障碍访问
- 国际化支持（中/英）

## 九、附录

### 9.1 竞品分析

| 产品 | 优势 | 劣势 |
|-----|------|------|
| 讯飞有声 | 音质好、音色多 | 非专注笔记场景 |
| 懒人听书 | 内容丰富 | 非个人笔记转换 |
| Speechify | 功能强大 | 中文支持弱、价格高 |
| 听书宝 | 免费资源多 | 非笔记转语音 |

### 9.2 核心差异化

1. **专注笔记场景**：深度适配笔记内容结构
2. **智能分段**：自动识别标题、段落，优化播报节奏
3. **学习增强**：重点内容重复播放、章节书签
4. **生态打通**：主流笔记平台一键同步
